# 🛡️ Diagnosta: El Guardián Inteligente para tus APIs

> **Análisis proactivo y diagnóstico de salud digital en tiempo real.**

Diagnosta no es un monitor convencional; es una capa de inteligencia que interpreta el estado de tus sistemas, detecta patrones de fallo y te ofrece soluciones en lenguaje humano antes de que el problema escale.

### ✨ Slogan
*Predice el síntoma, evita la caída de tus APIs.*

---

## 🌍 Infraestructura (CubePath + Coolify)
Desplegado sobre un **VPS GP Nano** en el datacenter de **Barcelona** para garantizar mínima latencia y máxima soberanía de datos.

## 🧠 Core Tecnológico
- **Orquestación:** n8n (Cerebro de IA).
- **Base de Datos:** PostgreSQL (Histórico de salud).
- **Despliegue:** Coolify (CI/CD continuo).
- **Infraestructura:** CubePath.
- **Acceso:** [https://diagnosta.botgrilo.es](https://diagnosta.botgrilo.es)
